# components

## Getting Started

Install dependencies,

```bash
$ tnpm i
```

Start the dev server,

```bash
$ tnpm start
```

Build documentation,

```bash
$ tnpm run docs:build
```

Run test,

```bash
$ tnpm test
```

Build library via `father-build`,

```bash
$ tnpm run build
```
